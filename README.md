# Chat Rocketseat API

This API was made with Fastify.

ORM: Prisma
Dependency of auth: Passport, JWT

Instructions to run in your machine:

Install dependencies:
```
yarn
```

Generate:
```
npx prisma generate
```

Migrate:
```
npx prisma migrate dev
```

Dev:
```
yarn dev
```

Build
```
yarn build
```

Run:
```
yarn start
```
